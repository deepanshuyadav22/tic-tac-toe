Rules of the Game :
    1. The game is played on a grid that's three squares by three squares.
    2. If one player is X, then second player is O. Players take turns putting their marks in empty squares.
    3. The first player to get three of his/her marks in a row (up, down, across OR diagonally) is the winner.
    4. When all nine squares are full, the game is over. If no player has three marks in a row, the game ends in a tie.

When to use Replay button :
    If the game is won by a player OR if the game is tied OR if you want to replay the game while playing without clearing scores (i.e. without setting scores to zero), then you have to click the Replay button.

When to use Restart button :
    If the game is won by a player OR if the game is tied OR if you want to restart the game while playing with clearing scores (i.e. setting scores to zero), then you have to click the Restart button.
